What is Markdown?

Markdown is a lightweight markup language that you can use to add formatting elements to plaintext text documents. Created by John Gruber in 2004, Markdown is now one of the world's most popular markup languages. Using Markdown is different than using a WYSIWYG editor. In an application like Microsoft Word, you click buttons to format words and phrases, and the changes are visible immediately.

GitHub Flavored Markdown (GFM)

GitHub Flavored Markdown, often shortened to GFM, is the dialect of Markdown that is currently supported for user content on GitHub.com and GitHub Enterprise. This tool supports popular GFM extensions such as:

  • Tables: Create tables with columns and rows using pipes | and hyphens -.
  • Task Lists: Create interactive checklists using - [ ] for an unchecked box and - [x] for a checked box.
  • Fenced Code Blocks: Add blocks of code with syntax highlighting using triple backticks ```.

Client-Side Processing

This Markdown preview tool runs entirely in your web browser using vanilla JavaScript. When you type or paste text into the editor, the conversion to HTML happens locally on your device. Your notes and data are never sent to a server, ensuring complete privacy, zero latency, and allowing the tool to work fully offline after the initial page load.
