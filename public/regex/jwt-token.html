Regex for Jwt Token

A regular expression to match a Jwt Token.

Regular Expression 
^[A-Za-z0-9-_]+\.[A-Za-z0-9-_]+\.[A-Za-z0-9-_]+$g

Explanation

Part Description
^ Start of the string
.* Matches any character
$ End of the string

Examples

  • ✓ eyJhbG.eyJzdWI.SflKxw
  • ✗ not.a.token!
