What is an XML Sitemap?

An XML sitemap is a file that lists a website's essential pages, making sure search engines like Google and Bing can find and crawl them all. It helps search engines understand your website's structure, which pages are most important, and when they were last updated.

Sitemap Tags Explained

  • <urlset>: The required root tag of the sitemap file.
  • <url>: The parent tag for each URL entry.
  • <loc>: The absolute URL of the page. This is the only required tag for each entry.
  • <lastmod>: The date the file was last modified, usually in YYYY-MM-DD format.
  • <changefreq>: How frequently the page is likely to change. This provides a hint to search engines, though they may not always adhere to it.
  • <priority>: The priority of this URL relative to other URLs on your site, ranging from 0.0 to 1.0. The default is 0.5.

Best Practices

  • Only include URLs that you want search engines to index. Exclude private pages, utility pages, or pages blocked by robots.txt.
  • Keep your sitemaps under 50MB and 50,000 URLs. If you have more, split them into multiple sitemaps and use a sitemap index file.
  • Submit your sitemap directly to search engines via Google Search Console or Bing Webmaster Tools.
  • Update the <lastmod> tag only when the page's content has changed significantly.
