Owner full access, Group read and execute, Others no permissions

What does chmod 750 do?

The permission 750 grants specific access rights to three different groups of users:

Owner (User)
Read (r) ✓
Write (w) ✓
Execute (x) ✓
Group
Read (r) ✓
Write (w) ✗
Execute (x) ✓
Others (Public)
Read (r) ✗
Write (w) ✗
Execute (x) ✗

When to use chmod 750

A specific configuration where Owner has full access, group and others have varying access (50).

Ready-to-copy Commands

chmod 750 filename
chmod -R 750 directory/
find . -type f -exec chmod 750 {} +
