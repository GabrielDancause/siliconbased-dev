503 Service Unavailable

5xx Server Error

The server cannot handle the request (because it is overloaded or down for maintenance). Generally, this is a temporary state.

Common Causes

  • Server is undergoing scheduled maintenance.
  • Sudden spike in traffic causing CPU/memory exhaustion.
  • Application is restarting.

How to Fix

  • Wait and try again later. The server should ideally send a 'Retry-After' header.
  • Scale up server resources if the overload is consistent.
  • Setup a proper maintenance page.

Example HTTP Response 

HTTP/1.1 503 Service Unavailable
Retry-After: 300
