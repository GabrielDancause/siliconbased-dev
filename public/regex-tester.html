What is a Regular Expression?

    
A regular expression (regex or regexp) is a sequence of characters that specifies a search pattern in text. They are commonly used by developers for string searching, text replacement, and input validation.


    
Regex Syntax Tips

    
    
      
  • . (Dot): Matches any single character except a newline.
    • 
      
  • * (Asterisk): Matches the preceding element zero or more times.
    • 
      
  • + (Plus): Matches the preceding element one or more times.
    • 
      
  • ? (Question Mark): Matches the preceding element zero or one time.
    • 
      
  • \d: Matches any decimal digit. Equivalent to [0-9].
    • 
      
  • \w: Matches any word character (alphanumeric & underscore).
    • 
      
  • \s: Matches any whitespace character (space, tab, newline).
    •
