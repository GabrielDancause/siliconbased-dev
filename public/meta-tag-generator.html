What are Meta Tags?

Meta tags are snippets of text that describe a page's content; the meta tags don't appear on the page itself, but only in the page's source code. Meta tags are essentially little content descriptors that help tell search engines what a web page is about.

Open Graph & Twitter Cards

The Open Graph protocol enables any web page to become a rich object in a social graph. It is used by Facebook, LinkedIn, Discord, and many others to display a preview card when your link is shared.

  • og:title: The title of your object as it should appear within the graph.
  • og:image: An image URL which should represent your object within the graph (recommended 1200x630 pixels).
  • og:description: A one to two sentence description of your object.

Twitter Cards work similarly but use `twitter:` prefixed properties. However, Twitter will fall back to Open Graph tags if specific Twitter tags are missing.

Best Practices for SEO

  • Title Tag: Keep it under 60 characters so it isn't truncated in search results. Make it descriptive and keyword-rich.
  • Meta Description: Keep it between 150-160 characters. Write compelling copy that encourages users to click through from search results.
  • Canonical URLs: Always include a canonical URL to prevent duplicate content issues if your page can be accessed via multiple URLs.
  • Images: Ensure your Open Graph image is high quality and properly sized so it looks great when shared on social media platforms.
